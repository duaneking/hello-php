Installation
------------

* Make sure you have installed gcc, php-dev ... These two should pull with them all dependencies needed.

* Run commands 
    * $ cd hello_php
    * $ phpize
    * $ ./configure
    * $ make
    * $ sudo make install
    * $ php -d extension=hello_php.so -r 'hello_php();'

* Now let's make some extensions!!!
